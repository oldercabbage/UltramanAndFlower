# 部署检查清单

部署前请检查以下项目：

## ✅ 文件检查

- [ ] `index.html` 存在
- [ ] `css/style.css` 存在
- [ ] `js/main.js` 存在
- [ ] `images/og-image.jpg` 存在（微信分享图片）
- [ ] 所有文件路径使用相对路径

## ✅ 功能检查

- [ ] 本地测试正常（运行 `./start.sh`）
- [ ] 动画播放正常
- [ ] 分享功能正常
- [ ] 响应式设计正常（手机/平板/电脑）

## ✅ 部署前准备

- [ ] 已创建GitHub账号
- [ ] 已创建GitHub仓库
- [ ] 已配置Git（用户名和邮箱）

## ✅ 部署步骤

- [ ] 运行 `./deploy.sh` 或手动推送代码
- [ ] 在GitHub仓库Settings中启用Pages
- [ ] 等待1-2分钟让GitHub处理
- [ ] 访问网站测试

## ✅ 部署后检查

- [ ] 网站可以访问
- [ ] 所有图片正常显示
- [ ] 动画正常播放
- [ ] 在手机浏览器测试
- [ ] 在微信中测试分享

## ✅ 微信分享配置

- [ ] 更新 `index.html` 中的 `og:url`
- [ ] 更新 `index.html` 中的 `og:image`
- [ ] 使用完整HTTPS URL
- [ ] 在微信中测试分享卡片

## ✅ 优化（可选）

- [ ] 压缩图片文件
- [ ] 添加自定义域名
- [ ] 配置Google Analytics（可选）

---

**完成所有检查后，您的网站就可以正常访问了！** 🎉
