python3 -m http.server 8000

### 4. 微信分享

1. 部署到GitHub Pages
2. 获取网站URL
3. 在微信中打开链接
4. 分享给朋友

## 使用说明

### 基本使用

1. 打开 `index.html` 或在浏览器中访问
2. 点击"播放动画"按钮
3. 观看送花动画
4. 点击"分享给朋友"分享链接

### 自定义配置

1. **修改祝福语**
   - 编辑 `index.html` 中的消息部分

2. **修改颜色**
   - 编辑 `css/style.css` 中的颜色值

3. **添加音乐**
   - 将音乐文件放入 `audio/` 目录
   - 在 `index.html` 中取消注释音频标签

4. **修改分享图片**
   - 运行 `python3 create_og_image.py`
   - 或使用设计软件创建 `images/og-image.jpg`

## 故障排除

### 动画不播放

- 检查浏览器控制台是否有错误
- 检查JavaScript文件是否加载
- 尝试刷新页面

### 分享不工作

- 确保使用HTTPS
- 检查og-image.jpg是否存在
- 检查Open Graph标签

### 移动端显示异常

- 检查viewport meta标签
- 清除浏览器缓存
- 检查CSS媒体查询
