# 图片资源目录

## 需要的图片

### og-image.jpg（必需）

微信分享时显示的图片：

- **尺寸**: 1200x630像素
- **格式**: JPG或PNG
- **用途**: 微信分享预览图
- **说明**: 建议包含贺卡的主要元素（奥特曼和花朵）

### 如何创建分享图片

1. 使用图片编辑软件（如Photoshop、GIMP等）
2. 创建1200x630像素的画布
3. 添加贺卡截图或设计图
4. 添加文字说明
5. 保存为JPG格式
6. 文件大小建议小于200KB

### 在线工具推荐

- Canva: https://www.canva.com
- Figma: https://www.figma.com
- 稿定设计: https://www.gaoding.com

## 其他图片（可选）

可以添加以下图片来增强效果：

- `background.jpg` - 自定义背景图片
- `ultraman.png` - 奥特曼图片（如果使用图片代替CSS绘制）
- `flower.png` - 花朵图片（如果使用图片代替CSS绘制）

## 注意事项

- 图片文件不要太大，建议每个文件小于500KB
- 使用适当的图片格式（JPG用于照片，PNG用于透明图片）
- 优化图片以减少加载时间
